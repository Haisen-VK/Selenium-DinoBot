# Selenium Based Chrome Dino Bot
A simple Selenium bot that can play Chrome Dino game

## Setting Up:


- In The .env file set `DRIVERPATH` to your chromedriver path
 ```
DRIVERPATH  = [./chromedriverpath]
```
